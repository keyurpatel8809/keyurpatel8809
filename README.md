## Featured Projects

### 🕒 Post-drop.app – Social Media Scheduling Tool
A SaaS platform for scheduling and automating social media posts for creators, startups, and small businesses.  
- Built with: React/TypeScript, Node, Express.js, supabase, AWS S3 cloud-hosted backend and database  
- Focus: Queue-based scheduling, multi-account support, clean UX, and analytics-friendly data model  

### 🏦 Banking & Fintech Microservices (CIBC / TechnoKwik)
Microservices and secure APIs for payments, accounts, and transaction processing in banking systems.  
- Built with: Java, Spring Boot, Spring Security, REST APIs, Oracle/MySQL, Kafka/RabbitMQ  
- Highlights: Transaction validation, audit trails, production support, and performance tuning in regulated environments  

### 📊 AI-assisted Banking Microservice
An AI-enabled microservice that uses a locally hosted LLM (via Ollama) to analyze logs and policy rules for banking queries.  
- Built with: Spring Boot, LLM integration, prompt tuning, secure on-prem deployment  
- Focus: Explainable responses, context-aware financial insights, and decision support for internal teams  

### 🌐 Full-stack Applications (GVT Software Solutions)
Full-stack Java + React applications with RESTful APIs, responsive UI, and production-ready code.  
- Built with: Java, Spring Boot, React, JPA/Hibernate, SQL databases  
- Focus: Clean API design, test coverage with JUnit/Mockito/Jest, and performance optimization
